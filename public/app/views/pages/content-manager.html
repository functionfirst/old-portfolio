Content Manager shatteredscales.co.uk

View website

An elegant content management system, designed to be simple to interactive with using a minimal UI. The architecture itself was based around a MongoDB document database accessed using an Express/MongooseJS Rest API.

Features

  • Inline page editing
  • Drag and drop image upload
  • Simple page creation
  • Auto-saving
  • Private/Public pages
  • Blog system
  • Universal commenting system for Blogs & Pages

Summary

Inline editing allowed an administrator to simply click on the live page and start editing. A built in contextual toolbar provided simple text formatting

To create a page simply highlight some text and type a link to a non-existent page, click the link you just created, then click 'Create'.
