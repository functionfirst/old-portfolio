Prototypes

Various prototypes developed over a number of years. These were designed to be additional modules added to the Claromentis framework which brought additional specialised functionality to the system.

Pages (2016)

View Pages prototype

Initial prototype for a new Claromentis module. The Pages module was designed to replace an existing but outdated CMS module. Pages was designed to be a plug-and-play system that allowed components to be dragged into place on the page and configured in place.

Timesheets - Prototype (2014)

View Timesheets protototype

Low-fidelity prototype for a Timesheet module based on the Claromentis framework. Built using Axure. Prototyping of a Timesheet module to extend the features available in within the Claromentis intranet framework.

Built using Axure, this prototype explored the required features that would allow Project Managers to manage a teams schedule as well as allowing the team to record their actual activity.
