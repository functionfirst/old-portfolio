The release of Claromentis 7.0 marked the culmination of months of development and collaboration by Claromentis' core product team.

The main focus of this release was to modernise an out-dated table-based layout by re-architecting and optimising over 1,000 html templates using the latest technologies and techniques.

Templates & Toolkits used:

  • HTML5 Boilerplate
  • Twitter Bootstrap (with responsive layout)
  • jQuery
  • Modernizr
  • Backbone.js

Document Preview

As part of the Claromentis 7.0 release, the DMS (Document Management System) was re-written using Backbone.js and a newly implemented RESTful web API.

This release also saw the introduction of document preview directly in the browser, allowing users to read the content of common files without having to download them.

Benefits of Claromentis 7.0

After the release of 7.0 the power and flexibility of the new interface has meant Claromentis has seen a marked increase in interest from new clients. The new Claromentis platform is:

  • Faster and easier to build custom interfaces and applications.
  • More uniform in design providing consistency and familiarity.
  • Supportive of a range of smartphones and tablets.
  • More competitive with powerful new features such as Document Preview.
